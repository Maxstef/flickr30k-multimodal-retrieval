# flickr30k-multimodal-retrieval

A machine learning project exploring image-text matching and multimodal retrieval using the Flickr30k dataset.

## Project Overview

The goal of this project is to investigate how well machine learning models can understand the relationship between images and natural language descriptions.

The project progresses through three stages:

1. **Binary Image-Text Matching** – determine whether a caption correctly describes an image.
2. **Contrastive Learning (Mini-CLIP)** – learn a shared embedding space for images and text.
3. **Retrieval & Analysis** – retrieve relevant captions for images (and vice versa) and analyze model performance.

## Dataset

* Flickr30k
* ~31,000 images
* ~159,000 human-written captions
* 5 captions per image

## Planned Models

* Logistic Regression baseline
* Neural Image-Text Matching model
* Mini-CLIP style contrastive model
